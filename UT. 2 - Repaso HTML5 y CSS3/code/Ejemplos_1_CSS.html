El selector universal * añade un color al fondo

  • El selector * es "peligroso" y muy potente ya que aplica estilos a TODOS los elementos del documento.
  • Con el combinador hermano general h1 ~ ul es decir que h1 y ul estan al mismo nivel ponemos en itálica estas 2 listas que siguen al h1.

En esta sección veremos el combinador descendiente.

  1. Aplicamos a la lista numerada, el combinador descendiente (_), desde la clase .parrafos1 hasta la clase <ol>, cambiando el color del texto.
  2. A esta línea, también se le aplicará el combinador descendiente ya que el estilo se aplica a todos los <li> dentro de la clase .parrafos1.
  3. Si lo dejamos tal cual veremos que heredamos un estilo sobre <li> que hemos definido anteriormente así que podremos otro estilo para sobreescribirlo.
